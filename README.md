# Automated PC Health Check

## Overview
This project demonstrates a PowerShell script that automates basic PC health checks for IT support purposes.  
It runs diagnostics for internet connectivity, disk space, CPU/memory usage, and startup programs.

## Features
- Internet connectivity test (ping + DNS)
- Disk space usage summary
- CPU and memory load check
- Startup program listing
